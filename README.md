# fulan-tech

> A Vue.js project

## Build Setup

``` bash
# install dependencies

1.依赖安装
npm install

# serve with hot reload at localhost:8082

2.项目运行
npm run dev

# build for production with minification

3.项目打包
npm run build

# build for production and view the bundle analyzer report
npm run build --report

4.测试
# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
